## Ответы на контрольные вопросы:
Как объявить структуру в языке С
1. Для объявления структуры используется ключевое слово struct.
struct Student {
    char name[50];
    int age;
    float grade;
};
2. Что такое вложенные структуры и зачем они нужны
Вложенные структуры - это структуры, которые содержат другие структуры в качестве своих полей. Они нужны для логической группировки связанных данных.
struct Date {
    int day;
    int month;
    int years;
};
Основная структура студента с вложенной структурой
struct Person {
    char name[50];
    struct Date birthDate;
};
3. Как передать структуру в функцию
Структуру можно передать в функцию двумя основными способами:
1) По значению (by value): Передается копия всей структуры.
void printStudent (struct Student s) {
    printf( "Name: %s, Age: %d\n", s.name, s.age);
    }
2) По указателю (by pointer): Передается адрес структуры.
void updateAge(struct Student *s, int newAge) {
    s->age = newAge;
    }
4. Можно ли создавать массивы структур
Да, можно.
struct Student students[5];
5. Как обратиться к элементу массива структур
Используя индексацию массива и оператор точки.
printf ("Имя первого студента: s\n", students[0]. name);
printf("Балл второго студента: %.2f\n", students[1].average_grade);
6. Можно ли использовать внутри структуры
Да, можно.
enum Course {first, seconf }:
struct Student {
    char name[50];
    int age;
    enum Course course;
    float average_grade;
};
7. Можно ли сравнивать структуры оператором ==
Нет, напряную сравнивать структуры оператором == нельзя. Компилятор выдаст ошибку.
Оператор == не определён для составных типов данных. Компилятор не знает, как сравнить все поля структуры одновременно.
