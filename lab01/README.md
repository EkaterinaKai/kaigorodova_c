# Лабораторная работа №1: Типы данных, память и выражения в C

## Цель работы
Научиться работать с базовыми типами данных языка C, понимать связь переменных и памяти, использовать операции и выражения, выполнять ввод и вывод данных.

## Задания

### Задание 1-3: Базовые типы и память
- Объявление переменных int, float, char
- Использование sizeof() для определения размеров типов
- Вывод адресов переменных в памяти (`%p`)

### Задание 4-6: Операторы
- Остаток от деления %
- Префиксный/постфиксный инкремент/декремент
- Приведение типов для деления

### Задание 7-8: Особенности вычислений
- Float неточность: 0.1f + 0.2f ≠ 0.3f из-за двоичного представления
- Переполнение int: 2147483647 + 1 = -2147483648 (двухдополнительный код)

### Задание 9-14: Практические задачи
- Вычисление алгебраических выражений
- Обмен переменных через временную
- Модульная арифметика для цифр числа
- Конвертация секунд в формат времени
- Анализ расположения переменных в памяти
